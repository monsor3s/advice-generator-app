## ♻ How to Contribute
```bash
  # Clone the project
  $ git clone https://github.com/monsor3s/advice-generator-app.git
```
```bash
  # Enter directory
  $ cd advice-generator-app
```
```bash
  # Install the dependencies, if use npm
  $ npm install
  $ npm install -D sass
```
```bash
  # Install the dependencies, if use yarn
  $ yarn install
  $ npm install -D sass
```
